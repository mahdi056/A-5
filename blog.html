Important FAQ


What is Document Object Model(DOM)?

Document object model is like a tree structure. It allows developers to access elements using javascript.

How do you select an element from the DOM?

I can select by using id, query selector and class.

What is event delegation in the context of the DOM and why it's useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners on individual child elements.It's useful for managing dynamic content, as it automatically applies to new child elements added to the parent without needing to reassign listeners.

How do you manpulate an elements attributes and styles using the DOM?
